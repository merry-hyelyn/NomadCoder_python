# 1. Lists in Python

### 1. sequence type

-   list(mutable)

-   tuple(immutable)

### 2. list

-   많은 value 를 나열하여 저장
-   선언방법 : []
-   콤마(,)를 이용하여 value 구별
-   index(value의 위치 값)는 **0**부터 시작

### 3. list common operation

-   "~" in [list] : list 안에 '~'요소가 있으면 True
-   len(list) : list의 길이 반환
-   list[index] : list를 index로 접근하여 원하는 위치의 요소 가져옴

### 4. list mutable operation

**mutable : 값 변경 가능**

-   list.append("~") : 요소 추가
-   list.remove("~") ; 요소 삭제
-   list.reverse : list를 역으로 출력
